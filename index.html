Smart-Glove for Fist Dance

Project Overview

This project tackles real-time hand gesture prediction to create immersive rhythm-based interactive experiences. We developed Fist Dance, a music-driven game combining "Rock, Paper, Scissors"-like gestures with directional movements.

The system features a wearable glove with five flex sensors and an IMU to capture finger positions and hand orientation. A dataset of over 80,000 samples from 10 users was collected and labeled using a beat-aligned sliding window method. A three-layer LSTM model was trained in stages: large-scale pretraining, user-specific finetuning, and hardware adaptation.
